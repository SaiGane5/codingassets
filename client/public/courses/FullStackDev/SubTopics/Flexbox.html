Flexbox

Flexbox (Flexible Box Layout) is a CSS module designed to create flexible and responsive layouts easily. It’s one-dimensional, meaning it arranges items in a row (horizontal) or column (vertical) with precise control over alignment, distribution, and spacing.

Let’s go through Flexbox properties and their professional usage in detail:

Basic Flexbox Setup

To use Flexbox, we first set display: flex on a container. This activates Flexbox for all its child elements.

.container {
    display: flex;
}

1. Flex Direction: flex-direction

Determines the direction of the main axis and, thus, the arrangement of flex items within the container.

  • Common Options:
    • row (default): Arranges items in a horizontal line, from left to right.
    • column: Arranges items in a vertical line, from top to bottom.
    • row-reverse: Like row, but items go from right to left.
    • column-reverse: Like column, but items go from bottom to top.

Professional Usage: Flex direction is often set to row (around 80% of cases) for navigation bars, button groups, and carousels, where a horizontal layout is standard. column is used in sidebars or vertical forms, where a vertical stack is more natural.

.container {
    display: flex;
    flex-direction: row; /* Can be column or reverse for alternative layouts */
}

2. Justify Content: justify-content

Aligns items along the main axis (horizontal if row, vertical if column).

  • Common Options:
    • flex-start: Items are aligned to the start of the container.
    • flex-end: Items are aligned to the end of the container.
    • center: Items are centered along the main axis.
    • space-between: Items are evenly distributed, with the first item at the start and the last at the end.
    • space-around: Items have equal space around them.
    • space-evenly: Items have equal space between them, including the edges of the container.

Professional Usage: justify-content: space-between and center are popular choices in about 50% of use cases. For example, in a navigation bar or a toolbar, space-between helps spread items evenly, whereas center is used for button groups or galleries to align items centrally.

.container {
    display: flex;
    justify-content: space-between;
}

3. Align Items: align-items

Aligns items along the cross-axis (perpendicular to the main axis).

  • Common Options:
    • flex-start: Items align to the start of the cross-axis.
    • flex-end: Items align to the end of the cross-axis.
    • center: Items are centered along the cross-axis.
    • baseline: Items align along their text baselines.
    • stretch (default): Items stretch to fill the container along the cross-axis.

Professional Usage: align-items: center is often used in professional layouts (around 70% of the time) because it neatly centers elements on the cross-axis, creating a visually balanced design. For full-width layouts, stretch is useful when we want items to extend across the container height or width.

.container {
    display: flex;
    align-items: center;
}

4. Align Content: align-content

Controls the spacing between wrapped rows or columns (only works when flex-wrap: wrap is applied).

  • Common Options:
    • flex-start: Rows are packed to the top of the container.
    • flex-end: Rows are packed to the bottom of the container.
    • center: Rows are centered within the container.
    • space-between, space-around, and space-evenly: Distributes space between rows.
    • stretch: Rows stretch to fill the container.

Professional Usage: align-content: center and space-around are common in grid-like layouts, such as image galleries or product listings, where items need alignment but also some space in between.

.container {
    display: flex;
    flex-wrap: wrap;
    align-content: center;
}

5. Flex Wrap: flex-wrap

Allows items to wrap onto multiple lines when they overflow the container’s width or height.

  • Options:
    • nowrap (default): All items are kept on a single line.
    • wrap: Items wrap onto multiple lines if needed.
    • wrap-reverse: Items wrap onto multiple lines in reverse order.

Professional Usage: Flex wrap is widely used (about 60% of the time) in layouts where content can expand, like responsive cards, image galleries, or product grids.

.container {
    display: flex;
    flex-wrap: wrap;
}

6. Order: order

Defines the order of items in the flex container. The default is 0, but items can have positive or negative values to adjust their order.

Professional Usage: order is typically used in responsive designs where layout priority changes on smaller screens (e.g., putting a logo first in a row or moving elements up/down).

.item1 {
    order: 2;
}

.item2 {
    order: 1;
}

7. Flex Grow: flex-grow

Controls how much an item can grow relative to the other items in the container. It takes a unitless value (0 is the default, which prevents items from growing).

Professional Usage: Commonly used in navigations or toolbars where a flexible item should take up remaining space.

.item {
    flex-grow: 1; /* Item will expand to fill the container space */
}

8. Flex Shrink: flex-shrink

Controls how much an item should shrink relative to others when there isn’t enough space. The default is 1, which allows items to shrink.

Professional Usage: It’s often set to 0 to prevent items from shrinking, commonly used in sidebars or fixed-size buttons within responsive designs.

.item {
    flex-shrink: 0; /* Prevents item from shrinking */
}

9. Flex Basis: flex-basis

Specifies the initial size of an item before the remaining space is distributed. It accepts values like pixels, percentages, or auto.

Professional Usage: flex-basis: 100px is a common use case for setting a minimum size for items, like cards or images in a responsive grid.

.item {
    flex-basis: 200px; /* Item takes 200px before flex-grow or flex-shrink adjustments */
}

10. Shorthand Property: flex

The flex shorthand combines flex-grow, flex-shrink, and flex-basis in one line for convenience.

  • Syntax: flex: [flex-grow] [flex-shrink] [flex-basis];

Professional Usage: This shorthand is heavily used for simplicity in professional layouts, especially when fine-tuning item size and flexibility.

.item {
    flex: 1 0 150px; /* Flex-grow: 1, Flex-shrink: 0, Flex-basis: 150px */
}

Flexbox is incredibly powerful and versatile, often used in more than 70% of professional, responsive web designs.
