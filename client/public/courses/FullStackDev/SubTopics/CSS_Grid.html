CSS Grid

CSS Grid is a two-dimensional layout system that allows for more complex and precise layouts by creating rows and columns within a grid container. With CSS Grid, you can design layouts that are both flexible and responsive.

Let's go through the main CSS Grid properties and how they’re used professionally with practical values and settings.

Basic CSS Grid Setup

To start using Grid, set display: grid on the container. This enables the grid layout for all child elements.

.container {
    display: grid;
}

1. Grid Template Columns & Rows: grid-template-columns and grid-template-rows

These properties define the columns and rows in a grid container. You can specify sizes in pixels, percentages, fr (fractional units), and more.

  • Common Options:
    • grid-template-columns: 1fr 2fr;: Creates two columns where the second column is twice as wide as the first.
    • grid-template-rows: 100px 200px;: Creates two rows, with the first row being 100px and the second 200px tall.

Professional Usage: Typically used to set grid layouts in portfolios, image galleries, or dashboards. The fr unit is particularly useful for responsive designs (about 60% of the time) because it distributes space evenly based on available area.

.container {
    display: grid;
    grid-template-columns: 1fr 2fr;
    grid-template-rows: 100px 200px;
}

2. Grid Gap: gap (also known as grid-gap)

Sets the spacing between rows and columns.

  • Common Options:
    • gap: 10px;: Adds 10px space between all rows and columns.
    • row-gap: 10px; column-gap: 20px;: Sets different gaps for rows and columns.

Professional Usage: The gap property is commonly used (in around 70% of designs) for cleaner layouts, particularly in card-based grids or photo galleries where spacing between items is essential for visual clarity.

.container {
    display: grid;
    gap: 20px;
}

3. Grid Auto Rows & Columns: grid-auto-rows and grid-auto-columns

Specifies the size of rows or columns that are auto-generated when content exceeds defined rows or columns.

  • Common Options:
    • grid-auto-rows: 100px;: Automatically sets additional rows to 100px.
    • grid-auto-columns: minmax(100px, auto);: Sets columns with a minimum width of 100px.

Professional Usage: Used in dynamic content grids where elements vary in size, such as blog or e-commerce listings. The minmax() function provides flexibility in responsive designs.

.container {
    display: grid;
    grid-auto-rows: 150px;
}

4. Grid Template Areas: grid-template-areas

Allows you to create named grid areas and place elements by referring to these names.

  • Example Usage:
    .container {
    display: grid;
    grid-template-areas:
        "header header"
        "sidebar main"
        "footer footer";
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
.footer { grid-area: footer; }

Professional Usage: grid-template-areas is highly readable and commonly used (in around 40% of layouts), especially in dashboard layouts, landing pages, and complex designs that require precise positioning.

5. Grid Column & Row Span: grid-column and grid-row

Defines how many columns or rows an item should span.

  • Common Options:
    • grid-column: 1 / 3;: Spans the item from column 1 to column 3.
    • grid-row: 2 / span 2;: Starts at row 2 and spans 2 rows.

Professional Usage: Used frequently (around 50%) in complex grid systems, like product grids, where items need to occupy multiple columns or rows, creating visual hierarchy.

.item {
    grid-column: 1 / span 2;
    grid-row: 1 / 3;
}

6. Justify Items and Align Items: justify-items and align-items

Aligns grid items within their cells along the inline (horizontal) and block (vertical) axes.

  • Common Options:
    • justify-items: center;: Centers items horizontally within their grid cells.
    • align-items: start;: Aligns items to the top within their cells.

Professional Usage: Around 70% of the time, these properties are used to align content within each grid cell, especially in card grids where content must be centered or aligned consistently.

.container {
    display: grid;
    justify-items: center;
    align-items: start;
}

7. Justify Content and Align Content: justify-content and align-content

Aligns the entire grid within the container along the inline (horizontal) and block (vertical) axes.

  • Common Options:
    • justify-content: space-around;: Distributes space around the entire grid along the horizontal axis.
    • align-content: center;: Centers the entire grid vertically within the container.

Professional Usage: Used mainly in responsive layouts where the grid needs to be centered or spaced consistently within a container, such as in a card layout for a portfolio.

.container {
    display: grid;
    justify-content: space-between;
    align-content: center;
}

8. Grid Auto Flow: grid-auto-flow

Controls the automatic placement of grid items when grid-template columns and rows are not explicitly defined.

  • Options:
    • row (default): Items fill rows first.
    • column: Items fill columns first.
    • dense: Fills all available spaces in the grid, useful for masonry-style layouts.

Professional Usage: Used in image or product grids where items of various sizes are “packed” efficiently, filling all available space.

.container {
    display: grid;
    grid-auto-flow: dense;
}

9. Minmax Function: minmax()

Defines minimum and maximum sizes for grid tracks (rows or columns).

  • Example:
    • grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));

Professional Usage: Frequently used in responsive designs (around 80%) for product or photo galleries where items need a minimum width but should expand as needed on larger screens.

.container {
    display: grid;
    grid-template-columns: repeat(auto-fill, minmax(150px, 1fr));
}

10. Repeat Function: repeat()

Defines repeating column or row tracks, simplifying grid definitions.

  • Example Usage:
    • grid-template-columns: repeat(3, 1fr);: Creates three equal columns.
    • grid-template-rows: repeat(auto-fit, minmax(100px, 1fr));: Creates rows that fit automatically based on available space.

Professional Usage: Often used for grids with repetitive elements, such as image or product listings, saving time and code for layouts with uniform sizing (used in about 75% of repetitive layouts).

.container {
    display: grid;
    grid-template-columns: repeat(4, 1fr);
}

Practical Example: Responsive Image Gallery

.gallery {
    display: grid;
    grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
    gap: 10px;
}

This setup creates a flexible, responsive grid where each item has a minimum width of 200px and expands as needed. The auto-fill feature helps add more columns as space allows, ensuring the layout adjusts seamlessly on different screen sizes.

CSS Grid is widely used for complex, two-dimensional layouts. In professional settings, it’s often applied to dashboards, photo galleries, and any content with both rows and columns. With properties like grid-template-areas and minmax(), it’s a versatile, powerful tool that works seamlessly alongside Flexbox for highly responsive and visually structured designs.
